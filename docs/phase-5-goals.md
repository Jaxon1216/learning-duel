# Phase 5：宏观目标模块

## 目标

在右侧内容区顶部显示用户的"精神宣言"，如"雅思7分"、"大厂Offer"。

## 设计理念

宏观目标不是任务，不可量化，没有进度条，没有截止日期。

它是一种**公开宣言**——告诉自习室里的其他人你在追什么。

类似于在桌面贴一张纸条。

## 组件：GoalBanner

显示为一排胶囊标签（pill badges）：

```
[雅思7分] [大厂Offer] [+ 添加目标]
```

- 最多 5 个（客户端限制）
- 自己的目标可以添加和删除
- 别人的目标只读

### 交互

点击 "+ 添加目标"：
1. 出现输入框
2. 输入标题，回车确认
3. 按 Escape 取消
4. 标签出现在列表中

点击目标旁的 ✕：
1. 直接删除（不弹确认框，因为目标很轻量）

## 数据结构

```
goals 表：
  id         → 自动生成
  user_id    → 默认 auth.uid()
  title      → 目标内容
  order_index → 排序
```

## 权限

- 所有人能看到所有人的目标（SELECT true）
- 只有自己能增删自己的目标

## 验收标准

- [x] 目标显示为一行胶囊标签
- [x] 最多 5 个
- [x] 能添加和删除自己的目标
- [x] 别人的目标只读，没有操作按钮
